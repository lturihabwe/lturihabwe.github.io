1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent.
//code fragment 1
$("li").each(function(idx, e) {
    $(e).css(“color”, “yellow”); });
//code fragment 2
$("li").each(function() {
    $(this).css(“color”, “yellow”); });
//code fragment 3
$("li").each(function(idx) {
    $(this).css(“color”, “yellow”); });

    Answer:
    The callback function inside the each method can have index and value parameter or not so the 3 code fragments are equivalent.
    In the first case, it loops through all the elements in the list and for each element e at index idx, it changes the css as shown.
    In the second case, it also loops through all the elements and changes css as shown above.
    And in the third case, it does the same.
