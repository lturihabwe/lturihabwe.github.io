jQuery Quiz I

1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?
Answer:
The JQuery function is the one whose job is to grab the elements you put into its parenthenses.
The JQuery object is what is returned when a JQuery function is invoked with a selector.


2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red.
Sample HTML:
<body>
<h1>abc</h1><br><div>
  <h1>div-1</h1>
  <h1>div-2</h1>
  </div>
  <h1>xyz</h1>
</body>

Answer:
$document.ready( function () {
    $(div > h1).css("background-color":"red");
});
